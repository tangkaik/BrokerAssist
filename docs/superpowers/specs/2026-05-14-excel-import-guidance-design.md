# Excel 导入要求提示设计

## 目标

用户点击「导入客户 Excel」后，先看到简洁明确的导入要求，再点击「选择 Excel 文件」继续选择文件。这样老用户随时能查看规则，但说明不常驻占用页面。

## 入口

移动端「我的」页保留原来的「导入客户 Excel」列表项。点击后不直接打开文件选择器，而是弹出底部说明 Sheet。

## 文案

Sheet 标题：`导入客户 Excel`

说明只保留 5 条：

1. 只支持 `.xlsx` 文件。
2. 第一行必须是表头。
3. 必须有「客户姓名」列，也可写「姓名」。
4. 可选列：电话、性别、年龄、生日、地址、标签。
5. 生日用 `YYYY-MM-DD`，标签可用 `、` 或 `,` 分隔；重复客户会跳过。

底部主按钮：`选择 Excel 文件`

次要按钮：`取消`

## 非目标

本次不做模板下载，不改 Web 端，不改后端导入规则。
