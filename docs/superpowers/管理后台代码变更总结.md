## 三、数据库变更

```sql
-- users 表新增字段
ALTER TABLE users ADD COLUMN is_admin BOOLEAN NOT NULL DEFAULT false;
ALTER TABLE users ADD COLUMN disabled BOOLEAN NOT NULL DEFAULT false;

-- 新表：系统配置
CREATE TABLE configs (
    id VARCHAR(36) PRIMARY KEY,
    key VARCHAR(100) UNIQUE NOT NULL,
    value TEXT NOT NULL,
    label VARCHAR(200),
    description TEXT,
    created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
);

-- 新表：行业定义
CREATE TABLE industries (
    id VARCHAR(36) PRIMARY KEY,
    key VARCHAR(40) UNIQUE NOT NULL,
    label VARCHAR(100) NOT NULL,
    role_name VARCHAR(100) NOT NULL,
    enabled BOOLEAN NOT NULL DEFAULT true,
    created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
);

-- 新表：行业提示词
CREATE TABLE industry_prompts (
    id VARCHAR(36) PRIMARY KEY,
    industry_key VARCHAR(40) NOT NULL REFERENCES industries(key) ON DELETE CASCADE,
    prompt_field VARCHAR(100) NOT NULL,
    value TEXT NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    UNIQUE(industry_key, prompt_field)
);
```

## 四、API 路由清单

所有路由均需 admin 权限，前缀 `/api/v1/admin`：

| 方法 | 路径 | 模块 |
|------|------|------|
| GET | `/stats` | 仪表盘统计 |
| GET | `/users` | 用户列表（分页+搜索） |
| PUT | `/users/{id}/password` | 重置用户密码 |
| PUT | `/users/{id}/status` | 启用/禁用用户 |
| GET | `/users/{id}/customers` | 查看用户客户 |
| GET | `/configs` | 配置列表 |
| PUT | `/configs/{key}` | 修改配置 |
| GET | `/industries` | 行业列表 |
| POST | `/industries` | 新增行业 |
| GET | `/industries/{key}` | 行业详情 |
| PUT | `/industries/{key}` | 编辑行业 |
| DELETE | `/industries/{key}` | 删除行业 |
| GET | `/industries/{key}/prompts` | 行业提示词 |
| PUT | `/industries/{key}/prompts` | 修改提示词 |

## 五、使用方式

1. 运行数据库迁移：`cd server && python -m alembic upgrade head`
2. 设置管理员：`UPDATE users SET is_admin = true WHERE account = '你的账号';`
3. 启动后端：`cd server && uvicorn app.main:app --port 8001`
4. 启动前端：`cd web && python3 dev_server.py`
5. 访问 `http://127.0.0.1:4173` 登录后，顶部栏出现「管理后台」链接
6. 或直接访问 `http://127.0.0.1:4173/admin.html`
